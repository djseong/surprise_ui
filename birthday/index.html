Instructions for the technologically illiterate

  1. Open terminal
  2. Run this command: curl -LJo surprise.zip https://github.com/djseong/surprise_script/zipball/master
  3. Run this command: unzip surprise.zip -d surprise1
  4. Run this command: cd surprise1
  5. If the previous command gave you something like cd: surprise1 No such file or directory, you goofed. Message dan
  6. Else, run this command: ls
  7. You should see one output, something like djseong-surprise_script-9aeedef (the numbers and letters at the end may be different)
  8. Type cd and the output you see, so like: cd djseong-surprise_script-9aeedef
  9. Run this command: python script.py
  10. If you exit the terminal and need to run the script again, start from step 4 again
